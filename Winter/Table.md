## **Base Assumption**

* **Source Price:** ₹50/bottle  
* **Bottle MRP:** ₹90 (set by manufacturer, within cap)  
* **You’re buying 100 bottles (₹5,000 total)**

---

## **🧾 Tiered Pricing Strategy**

| Channel | Buyer Type | Suggested Selling Price | Your Margin |  |
| :---- | :---- | :---- | :---- | :---- |
| **1\. Wholesalers / Retail Shops** | Middlemen (they resell) | ₹57 (max allowed) | ₹7 |  |
| **2\. Supermarkets / Local Stores** | Retail end sellers | ₹60 | ₹10 |  |
| **3\. Colleges / Institutions / Corporates** | End-user bulk (not resellers) | ₹70–₹75 | ₹20–₹25 |  |
| **4\. Online (Zepto / Blinkit / Swiggy Minis)** | B2C platform | ₹80–₹85 | ₹30–₹35 (before platform fee) |  |
| **5\. Combo Packs (Retail Offer Packs)** | Consumers (via local shops or stalls) | ₹85–₹90 | ₹35–₹40 |  |
| **6\. Custom Branding / Private Labeling (Later stage)** | Corporate orders (custom label) | ₹75–₹80 | ₹25–₹30 |  |
